These are some ARIA attributes to help screen readers

Aria attributes Purpose
aria-describedby Gives extra information regarding an element
aria-valuemax Tells the user what the max value could be
aria-selected = true/false Tells the user if it's true or false
aria-checked = true/false Tells the user what the max value could be
aria-hidden = true/false Hides an element from the user, assistive technology perspective
aria-expanded = true/false Tells the user if a region has become expanded or not
